\# DevOps Azure Project



\## 🚀 Project Overview

Deployed a Flask application on Azure VM using Terraform and configured NGINX as reverse proxy.



\## 🛠️ Tech Stack

\- Azure

\- Terraform

\- Linux (Ubuntu)

\- Python (Flask)

\- NGINX



\## 📂 Project Structure

\- app/ → Flask application

\- terraform/ → Infrastructure as Code

\- nginx/ → Reverse proxy config



\## ⚙️ Features

\- Automated infra provisioning

\- Secure VM access

\- Reverse proxy setup

\- Public IP access



\## 🌐 Output

Application accessible via public IP
